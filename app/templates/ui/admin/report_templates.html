Help / Guia de uso

Report Templates es un requisito transversal: no pertenece solo a Hoja de Cotejo. Todo reporte actual o futuro debe evaluarse contra este registro antes de cambiar headers, fotos, columnas o estructura.

  • No editar una plantilla base para resolver una propuesta nueva. Crear una version nueva.
  • No romper informes culminados: cada propuesta puede quedar amarrada a una version especifica.
  • Reportes sin asignacion: usan el comportamiento base por defecto.
  • Reportes futuros: deben agregarse a este catalogo y resolver su configuracion por report_key.
  • Cambios permitidos por version: header, logos/fotos, footer, columnas, etiquetas, orden y reglas visuales.
Nota tecnica: algunas pantallas ya pueden consultar la plantilla; otras quedan registradas aqui como requisito operativo para cuando se migren a plantillas versionadas.
